A Python-based network packet sniffer built using Scapy to capture and analyze live network traffic including IPs, protocols, and payload data.

# CodeAlpha - Network Sniffer

## 📌 Description
This project is a basic network packet sniffer built using Python and Scapy.

## ⚙️ Features
- Captures live network packets
- Displays source & destination IP
- Identifies protocols (TCP, UDP, ICMP)
- Shows packet payload data

## 🛠️ Technologies Used
- Python
- Scapy

## ▶️ How to Run
1. Install Scapy:
   pip install scapy

2. Run the script:
   python sniffer.py
